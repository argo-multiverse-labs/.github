# Argo Multiverse Labs

A collaborative space for **CNCF ecosystem demos, experiments, and conference presentations** related to ArgoCD, Argo Workflows, and cloud-native GitOps tooling.

## Conference Talks

| Talk | Conference | Demo Code |
|------|------------|-----------|
| [Everything, Everywhere All at Once: ApplicationSets & Matrix Generators](https://sched.co/1YFfG) | ArgoCon EU 2024 | [app-sets-project](https://github.com/argo-multiverse-labs/local-cluster/tree/main/app-sets-project) |
| [CD & Resource Health for SpinKube/WASM with ArgoCD](https://sched.co/1u5e1) | ArgoCon EU 2025 | [argo-spinkube-project](https://github.com/argo-multiverse-labs/local-cluster/tree/main/argo-spinkube-project) |

## Repositories

- **[local-cluster](https://github.com/argo-multiverse-labs/local-cluster)** — KinD-based demo environment with multiple conference projects
- **[argocd-example-apps](https://github.com/argo-multiverse-labs/argocd-example-apps)** — Comprehensive ArgoCD examples (Helm, Kustomize, Jsonnet, app-of-apps)
- **[workflows-demo](https://github.com/argo-multiverse-labs/workflows-demo)** — Argo Workflows sandbox

## Topics Covered

- ArgoCD & GitOps patterns
- ApplicationSets & Matrix Generators
- SpinKube / WebAssembly on Kubernetes
- Argo Workflows
- CD visualization & observability
- Multi-cluster deployments with vcluster

## Getting Started

Most demos use [KinD](https://kind.sigs.k8s.io/) for local Kubernetes clusters. Clone [local-cluster](https://github.com/argo-multiverse-labs/local-cluster) and follow the project-specific README for the demo you want to run.
